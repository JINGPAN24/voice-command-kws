项目简介
本实验基于传统 HTK 工具链，围绕语音识别中特征增强、多高斯混合建模及抗噪训练等核心问题，逐步优化语音识别系统性能。主要目标包括：

引入 Delta 与 Delta-Delta 特征，提升时序建模能力

对比单高斯与三高斯模型性能差异

探索多信噪比数据对识别结果的影响

分析混合训练集对抗噪性能的贡献

评估性别模型适配对特定语音识别效果的提升

实验任务与结果总结
📌 Task 1：引入 Delta 和 Delta-Delta 特征（MFCC_E_D_A）
操作内容：

在 config 文件夹中创建以 MFCC_E_D_A 结尾的配置文件

设置参数：

ini
复制
编辑
TARGETKIND = MFCC_E_D_A
DELTAWINDOW = 3
ACCWINDOW = 2
修改 ASR_LabDMML_MFCC_E_noCopy.pl，将路径替换为 MFCC_E_D_A

使用 s1d39 原型文件，设置 NUM_COEF = 39

性能比较结果：

特征类型	句子识别率 (%)	词识别率 (%)
MFCC_E	68.75	91.68
MFCC_E_D_A	88.02	97.57

结论：
引入 Delta 与 Delta-Delta 特征显著提升了识别准确率，尤其是词级准确率。通过调整 P 值，Delta 特征组合在 D/I（删除/插入）平衡性上也优于静态特征模型。

📌 Task 2：三高斯混合建模（GMM）
操作内容：

使用脚本 ASR_LabDMML_3mix_MFCC_E_noCopy.pl

设置参数：

perl
复制
编辑
$NUM_COEF = 39
$PAR_TYPE = "MFCC_E_D_A"
选择 E_D_A 配置文件及 39 维原型文件

性能提升：

模型类型	句子识别率 (%)	词识别率 (%)
单高斯模型	≈86.5	≈96.9
三高斯模型	≈88.0	≈97.6

结论：
虽然三高斯模型训练资源需求更高，但在语音建模准确性和误差均衡性方面均优于单高斯模型，尤其在复杂环境中效果更佳。

📌 Task 3a：低信噪比（SNR）环境测试
测试条件与结果：

环境	SNR	 句子识别率 (%)	词识别率 (%)	词准确率 (%)	插入错误 (I)	删除错误 (D)
N1  	10dB	68.75	        93.99	       90.75	28	4
N1  	15dB	81.25	        96.42	      94.45	17	1

结论：
随着 SNR 提升，句子与词识别率均明显改善，错误率下降，表明系统对噪声较敏感，特别在低 SNR 下性能下降显著。

📌 Task 3b：混合数据训练（Clean + Noisy）
实验对比：

训练集	测试 SNR	句子识别率 (%)	词识别率 (%)
Clean Only	Clean	90.10	98.15
Mixed	Clean	81.20	97.18
Clean Only	SNR10	68.75	93.99
Mixed	SNR10	81.20	97.18

结论：
混合训练在噪声条件下的鲁棒性更好，尤其在 SNR=10 情况下大幅超越纯净训练。但在纯净测试条件下稍有性能下降，表明混合训练存在一定“泛化妥协”。

📌 Task 4：女性语音适配模型评估
性能提升：

模型类型	句子识别率 (%)	词识别率 (%)
原系统	64.58	93.53
女性模型/混合模型	72.92	95.76

结论：
面向性别的语音建模可以明显提升女性语音的识别准确率，建议在部署阶段考虑性别适配的模型结构。
